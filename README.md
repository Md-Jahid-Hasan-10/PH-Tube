# 🎬 PH-Tube

PH-Tube is a dynamic video browsing web application built using **HTML, Tailwind CSS, and JavaScript**.
The application fetches video data from an API and dynamically displays it with interactive UI features such as category filtering, search functionality, modal views, and loading indicators.

This project demonstrates **API integration, DOM manipulation, conditional rendering, and dynamic UI updates using JavaScript**.

## ✨ Features

* 📡 **API Integration**
  Fetch and display video data from an external API.

* 📦 **Dynamic Data Rendering**
  Render arrays of objects dynamically on the webpage.

* 🔍 **Search Functionality**
  Search videos using API-based queries.

* 🧭 **Category Filtering**
  Filter videos by category and highlight the active button.

* ⚡ **Conditional Rendering**
  Use the **ternary operator** to handle conditional values.

* 📭 **Empty State Handling**
  Show a **"No Content Available"** message when the API returns empty data.

* 🪟 **Modal View**
  Display detailed video information inside a modal popup.

* ⏳ **Loading Spinner**
  Show a loading spinner while fetching data from the API.

* 🧠 **Logic-Based UI Updates**
  Implement interactive features using JavaScript logic.

## 📂 Project Structure

```
ph-tube/
│
├── assets/              # Images, icons, and logos
│
├── script/
│   └── index.js         # Core logic, API fetching, and DOM manipulation
│
├── index.html           # Main entry point of the application

└── README.md            # Project documentation
```

---

## 📁 Folder Explanation

* **assets/** → Contains images, icons, and other static resources used in the project.
* **script/** → Contains JavaScript files responsible for application logic.
* **index.js** → Handles API fetching, data rendering, search functionality, modal display, and UI updates.
* **index.html** → Main HTML file where the UI structure is defined.
* **tailwind.config.js** → Configuration file for Tailwind CSS.
* **README.md** → Documentation file describing the project.
